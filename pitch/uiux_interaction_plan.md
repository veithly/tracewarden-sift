## Screen Map

| Route | Screen | Primary user action | System response | State changed | Proof shown |
|---|---|---|---|---|---|
| `/cli` | Terminal evidence bench | Run `python -m tracewarden run ...` | Seals evidence, runs tools, verifies claims | Claim statuses and run artifacts | Terminal spans + run directory |
| `/receipts` | Claim receipt ledger | Open `claim_receipts.json` or HTML anchors | Shows each claim and evidence refs | Claim review state visible | File, line, hash, parser, tool call |

| `/dataset` | Dataset documentation | Open dataset docs | Shows case source, evidence files, ground truth | Reproducibility state visible | `docs/evidence-dataset.md` |

## First-Run Flow

- 0-10s: terminal starts the packaged case and prints evidence hash sealing.
- 10-30s: planner proposes suspicious claims and calls typed parsers.
- 30-60s: verifier challenges and revokes one false lead, then reroutes.
- 2-3min: judge opens reports and receipts; accuracy metrics are visible.

## Interaction Details

### `/cli` Terminal Evidence Bench

- Default state: project root with README command visible.
- Loading state: `[seal]`, `[plan]`, `[tool]`, `[verify]`, and `[write]` status lines stream.
- Empty state: missing evidence directory prints exact expected path and sample command.
- Error state: path guard violation or malformed evidence prints safe diagnostic and exits non-zero.
- Success state: run directory contains reports and terminal prints report paths.
- Keyboard/touch behavior: keyboard-only CLI; copyable commands; no pointer dependency.
- Accessibility note: terminal lines avoid color-only meaning; status words are textual.

### `/receipts` Claim Receipt Ledger

- Default state: list of receipt IDs grouped by status.
- Loading state: not applicable for static file; CLI prints write step.
- Empty state: no claims writes an empty receipt file with explanation.
- Error state: invalid receipt JSON fails tests and report generation.
- Success state: confirmed/revoked/challenged claims include evidence refs.
- Keyboard/touch behavior: HTML anchors support tab navigation.
- Accessibility note: status text is explicit and not color-only.

### `/accuracy` Accuracy Report

- Default state: metrics summary with confusion counts and known limitations.
- Loading state: CLI writes after verifier completes.
- Empty state: missing ground truth marks metrics unavailable and blocks "accuracy validated" copy.
- Error state: malformed ground truth exits non-zero with file path.
- Success state: precision, recall, F1, hallucination count, false positives, missed claims.
- Keyboard/touch behavior: report is printable and anchor-based.
- Accessibility note: tables include headings and plain text explanations.

### `/architecture` Trust Boundary Diagram

- Default state: SVG shows evidence root, typed tools, planner, verifier, reports.
- Loading state: static asset.
- Empty state: missing asset documented in README as blocking submission.
- Error state: CI checks file exists.
- Success state: judge can distinguish architectural guardrails from prompt rules.
- Keyboard/touch behavior: image has adjacent Markdown alt explanation.
- Accessibility note: `docs/architecture.md` mirrors the SVG text.

## Demo Choreography

- Judge input: one terminal command against `examples/evidence/case-alpha`.
- Live consequence: `CLAIM-004` moves from proposed to challenged to revoked.
- Proof artifact: `runs/demo/claim_receipts.json`, `execution_log.jsonl`, `accuracy_report.md`, `incident_report.html`.
- Fallback: `artifacts/sample-run/` generated by the same command during verification.

## Implementation Notes

- Components: `TraceWardenAgent`, `EvidenceToolset`, `ClaimVerifier`, `AccuracyScorer`, `ReportWriter`, `McpSchemaPrinter`.
- Data/API dependencies: local evidence logs, timeline JSON/CSV, ground truth JSON, generated run files.
- Test selectors: CLI stdout markers `[seal]`, `[plan]`, `[tool]`, `[verify]`, `[correct]`, `[write]`; HTML anchors `#claim-<id>`.
- Mobile QR behavior: QR points to static report/README; mobile reader taps receipt IDs and accuracy anchors.
